# 狼人杀游戏日志查看器

这是一个用于查看和分析狼人杀AI游戏日志的前端界面。

## 功能特性

- 📋 **日志列表**: 自动加载 `backend/data/game_logs` 目录下的所有日志文件
- 🔄 **实时刷新**: 支持手动刷新和自动刷新（5秒间隔）
- 📊 **游戏信息**: 显示游戏ID、开始时间、结束时间和游戏状态
- 👥 **玩家列表**: 展示所有玩家及其角色，标记已出局玩家
- 🎮 **回合详情**: 按回合展示夜晚和白天阶段的所有动作
- 💭 **详细信息**: 显示每个玩家的心声、表现和发言
- 🎨 **美观界面**: 现代化的渐变设计，响应式布局

## 使用方法

### 方式一：使用Python服务器（推荐）

1. 确保已安装Python 3.6+

2. 在项目根目录运行服务器：
```bash
python frontend/server.py
```

3. 打开浏览器访问：
```
http://localhost:8080
```

### 方式二：使用其他Web服务器

如果你有其他Web服务器（如nginx、Apache等），可以将 `frontend` 目录配置为静态文件目录，并确保API路由正确配置。

## 界面说明

### 顶部控制栏
- **日志选择器**: 下拉菜单选择要查看的日志文件（默认显示最新的）
- **刷新按钮**: 手动刷新当前日志内容
- **自动刷新**: 勾选后每5秒自动刷新一次（适合观看进行中的游戏）

### 游戏信息区
显示当前游戏的基本信息：
- 游戏ID
- 开始时间
- 结束时间（如果游戏已结束）
- 游戏状态（进行中/已结束）

### 玩家列表
以卡片形式展示所有玩家：
- 玩家名称
- 角色（狼人、村民、预言家、女巫、猎人）
- 存活状态（已出局的玩家会显示灰色并标记💀）

### 游戏回合
按时间顺序展示每个回合的详细内容：

#### 夜晚阶段 🌙
- 狼人讨论和投票
- 女巫使用药水
- 预言家查验身份
- 夜晚死亡公告

#### 白天阶段 ☀️
- 玩家公开发言
- 投票环节
- 投票结果
- 遗言
- 白天死亡公告

每个动作都包含：
- ⏰ 时间戳
- 👤 玩家名称
- 💭 心声（内心想法）
- 👁️ 表现（外在行为）
- 💬 发言（实际说的话）

## 角色说明

- 🐺 **狼人** (红色): 夜晚击杀村民，白天伪装身份
- 👨‍🌾 **村民** (绿色): 通过投票找出狼人
- 🔮 **预言家** (蓝色): 每晚可查验一名玩家身份
- 💊 **女巫** (紫色): 拥有解药和毒药各一瓶
- 🏹 **猎人** (橙色): 被淘汰时可开枪带走一名玩家

## 技术栈

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **后端**: Python 3 (http.server)
- **样式**: 渐变设计 + 响应式布局
- **数据**: 实时解析日志文件

## 浏览器兼容性

支持所有现代浏览器：
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+

## 注意事项

1. 确保 `backend/data/game_logs` 目录存在且包含日志文件
2. 日志文件必须是 `.log` 格式
3. 如果开启自动刷新，建议在游戏进行中使用
4. 大型日志文件可能需要较长加载时间

## 故障排除

### 无法加载日志列表
- 检查服务器是否正常运行
- 确认 `backend/data/game_logs` 目录存在
- 查看浏览器控制台是否有错误信息

### 日志内容显示不完整
- 确认日志文件格式正确
- 检查日志文件编码是否为UTF-8
- 尝试手动刷新

### 自动刷新不工作
- 确认已勾选"自动刷新"选项
- 检查浏览器是否阻止了定时器
- 尝试手动刷新后再开启自动刷新
